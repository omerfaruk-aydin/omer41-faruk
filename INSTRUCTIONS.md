# GitHub Profile Setup Instructions

This folder contains everything you need for your premium GitHub profile.

## Structure
- `README.md`: The main profile content with your links, stats, and 3D map placeholder.
- `.github/workflows/3d-map.yml`: The automation script that generates your 3D contribution map daily.

## How to use this elsewhere:
1. Create a repository on GitHub named `omer41-faruk`.
2. Push the contents of this folder to that repository.
3. Make sure the `.github` folder is at the root of the repository.
4. Go to the **Actions** tab in your repository settings and ensure Actions are enabled.
5. The 3D map will generate automatically every night, or you can trigger it manually from the Actions tab.

Enjoy your new profile! 🚀
